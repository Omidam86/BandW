# BandW

BandW یک بلاگ متن‌محور حرفه‌ای با طراحی مینیمال و رابط کاربری مدرن است. این پروژه روی خوانایی، سادگی و تجربه کاربری روان در موبایل و دسکتاپ متمرکز شده و محیطی خالص برای خواندن و نوشتن متن فراهم می‌آورد.

ویژگی‌های قابل مشاهده:
- طراحی تاریک و مینیمال با تاکید بر تایپوگرافی و فاصله‌گذاری
- فید مقالات با کارت‌های خلاصه و متادیتا (دسته، تعداد بازدید و …)
- صفحه ارسال مطلب و پنل مدیریت برای بازبینی و انتشار
- جستجوی ساده، پیش‌نمایش مقاله و نمایش مقالات مشابه
- واکنش‌گرایی برای دستگاه‌های مختلف و بهینه‌سازی رابط کاربری

این مخزن شامل نسخه‌هایی از صفحات رابط کاربری (برای توسعه داخلی و آزمایش) است و حاوی نمونه‌کدهای فرانت‌اند می‌باشد.

تمامی حقوق این مخزن محفوظ است. هرگونه کپی، نشر، توزیع یا استفادهٔ بدون اجازهٔ صریح صاحب پروژه ممنوع است.
